Ruby Loops VS. JavaScript Loops

For loops are used when we need to iterate a specific number of times. This type of loop is not often used in Ruby.
For each allows you to loops through each item in a collection.
While loops allow you to iterate while a certain condition or conditions hold true. Once false, the loop will terminate.

  • For loop -- Ruby for i in 0..5 puts "We are on number: #{i}" end
  • For loop -- JavaScript for (i = 0; i < 5; i++) { console.log("We are on number: " + i); }
  • Each method -- Ruby fruits = ["apples", "oranges", "blueberries", "bananas"] fruits.each do |fruit| puts "#{fruit} are a fruit." end
  • forEach function -- JavaScript fruits = ["apples", "oranges", "blueberries", "bananas"]; function printFruits(fruit) { console.log(fruit + " are a fruit"); } fruits.forEach(printFruits);
  • While loop -- Ruby count = 10 while count >= 0 puts count count -= 1 end
  • While loop -- JavaScript var count = 10 while (count >= 0) { console.log(count); count--; }
